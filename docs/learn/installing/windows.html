A good setup for Perl on Windows (32 and 64 bit):

  • Window tools
  • Make sure you do not have any version of Perl already installed. [%# (including from msys - but people might need for some other build systems %] (see video below for details), If you do uninstall it through the Window control panel "add/remove programs", if you still have a C:\Strawberry folder rename or delete it
  • Download and install Strawberry Perl. You also get many other useful CPAN modules as well).
  • Log out and back in (or reboot)
  • Go to your start menu, then click the "Perl command" link
  • To confirm the installation worked type: [% WRAPPER code_snippet %]perl -v[% END %]
  • Install App-cpanminus by typing in: [% WRAPPER code_snippet %]cpan App::cpanminus[% END %]

Installing Strawberry Perl (without Padre) video

Explanation of this setup

  • Strawberry Perl (a Perl packaged for Windows) is recommended as you get many useful modules (especially those that are tricky to install) along with it.
  • To interact with the command line and run Perl commands, you need to run 'cmd'. There are better alternatives available as well.
  • App-cpanminus provides the cpanm tool, which makes installing Perl modules very simple to do.
